# Cloud Monitoring Project

## Overview
This is a beginner-level project to understand system monitoring and cloud infrastructure basics.

## Features
- CPU usage monitoring
- Memory usage tracking
- Disk usage checking

## Technologies Used
- Linux 
- Bash scripting
- Cloud concepts (AWS)

## Description
This project simulates how cloud engineers monitor system performance using basic tools and scripts. It helps in understanding infrastructure and troubleshooting.

## Future Improvements
- Integrate with AWS EC2
- Add alert system
- Automate monitoring

## Project Architecture (Basic Idea)

User → Server (Linux) → Monitoring Script → System Metrics

## Cloud Relevance

This project simulates how cloud engineers monitor servers in AWS environments.
