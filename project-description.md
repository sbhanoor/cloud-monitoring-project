# Cloud Monitoring Project - Detailed Explanation

## Objective
To understand how cloud engineers monitor system performance in real-time.

## What I Learned
- Basics of cloud infrastructure (AWS EC2 concept)
- Linux system monitoring
- Importance of CPU, memory, and disk tracking
- Troubleshooting system performance issues

## Tools Used
- Linux
- Bash scripting
- Cloud concepts (AWS basics)

## How It Works
This script checks system resources:
- CPU usage
- Memory usage
- Disk usage

These are important in cloud environments to ensure system stability.

## Real-world Use
In cloud platforms like AWS, tools like CloudWatch monitor these metrics.

## Future Scope
- Integrate with AWS CloudWatch
- Automate alerts
- Add logging system
